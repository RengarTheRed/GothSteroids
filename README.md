# Asteroids – Goth Girls Edition

A fun Asteroids clone built with Python and Pygame where you shoot hearts at goth girl asteroids!

## Features

- Classic Asteroids-style gameplay with floaty, inertia-based movement
- Goth girl sprites as asteroids that drift and rotate
- Heart-shaped bullets
- Screen-wrap for all objects
- Start screen, pause functionality, and game over with restart option
- Simple scoring system

## Requirements

- Python 3.x
- Pygame

## Installation

1. Clone this repository
2. Create a virtual environment (optional but recommended):
   ```bash
   python3 -m venv venv
   source venv/bin/activate  # On macOS/Linux
   ```
3. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## How to Play

```bash
python3 main.py
```

### Controls

| Key | Action |
|-----|--------|
| ↑ Up Arrow | Thrust forward |
| ← Left Arrow | Rotate left |
| → Right Arrow | Rotate right |
| Space | Shoot hearts |
| ESC | Pause / Resume |
| R | Restart (on game over) |
| Q | Quit (on game over) |
